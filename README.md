# Brand Name Generator

A simple web app that helps entrepreneurs brainstorm brand names based on keywords and tone.

## Features
- Keyword input
- Tone selector (Fun, Luxury, Serious)
- Randomized brand name generation

## How to Use
1. Enter a keyword (e.g., "green", "tech", "zen")
2. Choose a tone
3. Click "Generate Name"
